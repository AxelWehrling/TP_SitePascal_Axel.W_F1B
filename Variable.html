Les Variables:

Les variables permettent de contenir quelque chose, ça peut être un entier, un mot, un caractère... Mais pour celà il est déjà néccessaire de leur attribué un type tel que:
-integer(entier),la variable ne peut que contenir un nombre entier.
-real(réel),la variable ne peut que contenir un nombre réel.
-string(chaine),la variable ne peut que contenir une chaîne de caractère.
-char(charactere),la variable ne peut que contenir un caractère.
-booleen(booléen),la variable ne peut que renvoyer une reponse "Vrai" ou "Faux".

On peut ensuite faire appel à cette variable dans sont programme pour utiliser son contenue. Il est tout a fais possible de modifier la valeur d'une variable directement dans le programme, ce qui n'est pas le cas d'une constante.

exemple de programme:

programm Variable;

uses crt;

var x,z:interger; y:string;

BEGIN
clrscr;
x:=8;
y:='jambon';
z:=x+8;
writeln(x,z,y);
END.
