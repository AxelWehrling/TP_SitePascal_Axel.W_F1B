Les Variables:

Les constantes ont la même propriété que les variables sauf qu'elles ne peuvent qu'être de type "integer" ou "real" et leur valeur ne peut pas être modifié dans le programme.Elles sont utiles pour les valeurs fréquemments et qu'on sait d'avance qui ne sera pas modifié. On initialise les constantes avant les variables:

exemple programme:

programm Constante;

uses crt;

const kek=8.8;

var x,z:interger; y:string;

BEGIN
clrscr;
x:=8;
y:='jambon';
z:=x+8+kek;
writeln(x,z,y,kek);
END.
